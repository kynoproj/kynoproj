# Contributing to the Kyno Project

Read and abide by the
[Kyno Project Code of Conduct](https://github.com/kynoproj/kynoproj/blob/master/CODE_OF_CONDUCT.md).

Kyno Project uses the DCO.

- https://github.com/apps/dco/

Please signoff your contributions by doing ONE of the following:

- Use `git commit -s ...` with each commit to add the signoff or
- Manually add a `Signed-off-by: Your Name <your.email@example.com>` to each
  commit message.

The email address must match your primary GitHub email. You do NOT need
cryptographic (e.g. gpg) signing.

- Use `git commit -s --amend ...` to add a signoff to the latest commit, if you
  forgot.
